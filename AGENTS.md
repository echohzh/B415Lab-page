# AGENTS.md

## 项目说明

本项目是一个纯静态实验室项目展示网站，用于展示 B415 课题组语音方向项目、论文成果和音频 Demo。

## 适用范围

本文件供各类自动化 Agent、代码助手或维护脚本参考。执行任务时应遵循本项目现有结构，不主动引入复杂工程化改造。

## 关键文件

- `index.html`：首页结构
- `demo.html`：项目 Demo 详情页结构
- `styles.css`：全站样式
- `script.js`：实验室信息、项目数据、论文数据和页面渲染逻辑
- `assets/`：音频、图片、第三方 Demo 页面等静态资源

## 修改规范

1. 内容更新优先修改 `script.js` 中的数据对象和数组。
2. 样式调整仅修改 `styles.css`，避免内联样式扩散。
3. 页面结构变更应保持 `index.html` 与 `demo.html` 的语义化和可访问性。
4. 新增资源应放入 `assets/` 下清晰命名的项目目录。
5. 修改资源路径时必须同步检查所有引用位置。

## 验证方式

推荐使用本地静态服务器验证：

```bash
python3 -m http.server 8000
```

检查首页、Demo 页面、搜索筛选、音频播放和外部链接。

## 禁止事项

- 不要提交 `.DS_Store`、缓存文件或编辑器临时文件。
- 不要随意更改项目 `id`。
- 不要删除仍被页面引用的资源。
- 不要在未获要求时引入 React、Vue、构建工具或包管理依赖。
- 不要把 Markdown 链接语法写入 HTML 正文，应使用 `<a href="...">...</a>`。

## 文案要求

保持中文为主、表述准确、学术展示风格统一。论文题目、模型名和仓库名保留原始英文大小写。
